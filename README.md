# cloud_computing

# 使用指南

本項目基於 Python，並依賴虛擬環境來管理依賴項。請按照以下步驟執行：

## 📂 克隆倉庫
```bash
git clone <repository-url>


cd combined
```
## 🚀 啟動虛擬環境
```bash
# macOS/Linux
source venv/bin/activate

# Windows
venv\Scripts\activate
```
## 📦 安裝依賴
```bash
pip install -r requirements.txt
```
## 🌐 啟動應用程式
```bash
python web.py
```
## ⏹️ 停止虛擬環境
```bash
deactivate
```
